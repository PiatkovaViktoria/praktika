# VK QUIZ MVP (локальный запуск)

Минималистичное веб-приложение для проведения квизов в реальном времени.

## Что реализовано

- Регистрация и авторизация пользователей (`участник` / `организатор`)
- Создание квизов организатором:
  - категории
  - время на вопрос
  - правила
  - вопросы с текстом/изображением
  - тип ответа: одиночный/множественный
- Запуск комнаты с кодом доступа
- Подключение участников по коду комнаты
- Демонстрация вопросов в реальном времени через WebSocket
- Приём ответов только во время активного показа вопроса
- Подсчёт баллов и лидерборд
- Личный кабинет:
  - организатор: список созданных квизов
  - участник: история прохождений

## Технологии

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Realtime: Socket.IO
- Хранение данных: JSON-файл (`data/db.json`) для локального MVP

## Запуск

```bash
npm install
npm start
```

Откройте: `http://localhost:3000`

## Как протестировать быстро

1. Зарегистрируйте организатора и создайте квиз.
2. Нажмите `Запустить` и получите код комнаты.
3. Во второй вкладке/браузере зарегистрируйте участника.
4. Введите код комнаты и подключитесь.
5. Организатор нажимает `Запустить квиз`.
6. Проверьте изменение лидерборда и сохранение истории после завершения.
